# Entendimento

Nessa primeira etapa, o grupo se reúne para definir as seguintes questões do projeto:

- Metas
- Métricas
- Riscos
- Como chegaremos no sonho
- Personas
- Descobertas
- Core (Matriz SWAT e Lean Inception)
